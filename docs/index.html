Tags e seus sentidos

Esse é um texto com uma parte em Negrito com a tag <strong>.

Esse é um texto com uma parte com ênfase com a tag <em>.

Podemos criar um texto marcado com a tag <mark>.

Podemos criar um texto pequeno com a tag <small>

Podemos marcar um texto como excluído com a tag <del>

Podemos marcar um texto como inserido com a tag <ins>

Podemos marcar um texto como sobrescrito como x2+2y com a tag <sup>

Podemos marcar um texto como subscrito como H2O com a tag <sub>

Também podemos colocar um trecho de código combinando as tags <pre>, que pré-formata o texto e <code>, que é específica para códigos.

                 
num = int(input('Digite um número'))
if num % 2 == 0
    print(f'O número {num} é par')
else
    print(f'O número {num} é impar')

Podemos colocar citações com a tag <q>

Também exitem as citações em blocos, em que usamos a tag <blockquote>, e ainda podemos colocar o link fonte no parâmetro "cite" da tag.

"It was a Big Big world but we tought we was bigger."

Podemos criar abreviações em HTML com a tag <abbr>, usando o parâmetro title para dizer o que significa a abreviação.
