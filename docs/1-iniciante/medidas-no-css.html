Elementos Relativos

em
Relativo ao tamanho da fonte do pai.
rem
Relativo ao tamanho da fonte do root.
vw
Relativo à 1% da largura da viewport.
vh
Relativo à 1% da altura da viewport.
%
Relativo ao elemento.

Funções

Calc
calc (100% - 40px) Pode ser usada para calcular 100% da vw e retirar 40px.

Element selector
Busca um elemento dentro do HTML, como <h1>.
Id selector
Busca pelo id, sendo representado pela '#'.
Class selector
Seleciona pela classe, usando o '.'.
Attribute selector
Busca um elemento que tem determinado atributo. É representado por [].

Combinators

Descendent Combinator
Busca um elemento que é descendente de outro, é representado pelo espaço ' '.
Child Combinator
Seleciona apenas o elemento que é filho direto do pai. É representado pelo ">"
Sibling Combinator
Seleciona o irmão direto (primeiro irmão) na hierarquia. Representado pelo '+'
General Sibling Combinator
Seleciona todos os elementos irmãos. É representado pelo '~'

Pseudo Class

Começam com dois pontos e representam um elemento que está em um estado específico.

nth-of-type
Pega dermidado elemento de um tipo, por exemplo: 'p:nth-of-type(1)', que pega o primeiro elemento 'p'
nth-of-child
Pega dermidado elemento filho do item principal.
Outra coisa bem legal é que podemos colocar 'odd' ou 'even' ao invés do número.
hover e focus
O primeiro é quando o mouse passa por cima e o segundo é quando um input está em foco.

Pseudo Element

before
Coloca um item antes de determinado conteúdo. Obs.: Sempre requer a propriedade 'content'.
after
Coloca um item depois de determinado conteúdo. Obs.: Sempre requer a propriedade 'content'.
first line
Pega somente a primeira linha.
