Começando pelo começo

Toda caixinha tem 'margin', 'padding', 'border' e 'content'

Box-sizing
Ele configura como será calculado o tamanho total da caixa. Pelo padrão, ele vem como 'content-box', que preferencia o conteúdo. Para respeitar sempre a borda, colocamos 'border-box'.
Diplay block & inline
Ele configura como será o comportamento das caixas em relação às outras. O parametro 'block' coloca as outras caixas para baixo. As <div> são display block, já o <span> é display inline. Temos que considerar que o block respeita parâmetros width e height, o inline não. Além disso, o inline não respeita valores verticais de'margin', 'padding' e 'border'
